Direct proportion — same direction
Double x → y also doubles. The graph is a straight line through the origin. x/y = k    (constant quotient) Examples: more time at fixed speed → more distance; more pencils → more cost.
Inverse proportion — opposite direction
Double x → y halves. The graph is a curve (hyperbola) that never touches the axes. x × y = k    (constant product) Examples: more workers → fewer days; faster speed → less travel time.
Quick test — how to tell which kind?
Direct If x doubles and y also doubles — it is direct. The ratio x:y stays the same.
Inverse If x doubles and y halves — it is inverse. The product x×y stays the same.
Direct The graph is a straight line starting at (0,0).
Inverse The graph is a curve bending away from both axes.
